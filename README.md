# Valuable Youtube Channels/Podcasts
## Author: @antero360


### Author Notes
This is an extensive list of the youtube channels which I have found value in over the years. I have divided this list into four general categories: 


**Software Development:** All things Software Engineering related.

**Tech General:** Anything from software reviews to hardware and tech news.

**Information/Cyber Security:** All things information privacy, cyber security, networking.

**Miscellaneous:** Non tech related topics that still affect tech


By no means is this list complete. This will be a breathing document, so expect frequent changes. If you would like to add to this list, feel free to create a PR. Please insert your contributions into the correct alphabetical placement and make sure to add the following line to the end of every item you add:

Contribution by: @githubUsername
